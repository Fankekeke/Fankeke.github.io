Angular路径别名配置

2018-12-27
0
Angular

angular cli 内置了 webpack,所以也是可以配置路径别名的. 而且配置的方式几乎是一样的.

配置路径别名

找到你的项目根目录的 tsconfig.json 文件.

注意: 如果你的配置不生效,需要查看你的baseUrl是否配置正确.

{
    "compilerOptions": {
        "baseUrl": "./src/", 
        "paths": {
            "@app/*": ["app/*"],
            "@services/*": ["app/services/*"]
        }
    }
}

配置之前

import { Api } from '../../../../../services/api.service';
import { xxx } from '../../../../../services/api.xxx';

配置之后

import { Api } from '@services/api.service';
import { xxx } from '@services/api.xxx';
